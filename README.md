# [Full Stack open](https://fullstackopen.com/en/) - University of Helsinki
### About the course
This course is maintained by the [University of Helsinki](https://www.helsinki.fi/en) and serves as an introduction to modern web application development with JavaScript. The main focus is on building single page applications with ReactJS that use REST APIs built with Node.js. It also contains a section on GraphQL, a modern alternative to REST APIs.\
The course covers: React, Redux, Node.js, MongoDB, GraphQL and TypeScript, testing, configuration and environment management, the use of databases for storing the application’s data amongst other things.

### Part 0 - Fundamentals of Web apps
This part covers general information about the course and fundamentals of web apps like principles of web development, HTTP requests, tradicional web applications, DOM, SPA...
